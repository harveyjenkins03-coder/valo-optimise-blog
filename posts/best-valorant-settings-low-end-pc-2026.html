  1. In AMD Software → Performance → Tuning

Section 05 Intel UHD — Squeeze Every Frame from Integrated Graphics

Playing on integrated graphics is the hardest scenario, but it's achievable. The key difference: Intel UHD shares system RAM as VRAM, so RAM speed and amount directly impacts your FPS more than anything else.

Intel Graphics Command Centre settings:

SettingValue
Application Optimal ModeOn
Anti-AliasingApplication Controlled
Anisotropic FilteringApplication Controlled
Vertical SyncApplication Controlled

Critical steps specific to Intel UHD:

  1. Enable XMP/EXPO in BIOS — Intel UHD gets 20–35% more FPS just from running RAM at rated speed (e.g. 3200MHz vs 2133MHz default). This is the single biggest upgrade for integrated graphics.
  2. Set VALORANT to 900p or 720p — integrated graphics cannot sustain 1080p at competitive framerates. Drop to 1280×720 for 60+ FPS. Use Section 6's method.
  3. Close everything — integrated graphics uses system RAM for VRAM. Chrome alone eats 500MB–2GB. Close all browsers and background apps before playing.
  4. Ensure your laptop is plugged in — unplugged, Intel UHD throttles to 40–60% power to save battery. Plugged in, you get full performance.
📈
UHD 620/630: realistic 55–70fps at 720p with these steps UHD 770 (12th gen): realistic 75–90fps at 720p Not ideal for ranked, but fully playable for learning and casual play.

Section 06 Resolution: The Single Biggest FPS Multiplier

Dropping your render resolution is the most powerful tool available to low-end players. It directly reduces the number of pixels your GPU processes per frame — and VALORANT's art style holds up well at lower resolutions compared to most games.

How to set a lower resolution in VALORANT:

  1. Settings → Video → General → Resolution → set to 1600×900
  2. Set Display Mode to Fullscreen (not Windowed Fullscreen — fullscreen gets GPU priority)
  3. If using AMD RSR: set Windows resolution to 1600×900 first, then set VALORANT to the same
💡
1600×900 on a 1080p monitor looks fine in VALORANT. The game's UI elements scale correctly and the character models remain clear at medium-close range. The tradeoff at longer ranges is minimal at Bronze–Plat level play.
Section 07 Windows Tweaks That Matter Most on Low-End Hardware

On low-end machines, Windows overhead takes up a larger percentage of total system resources. These tweaks free up CPU and RAM specifically for VALORANT.

1. Ultimate Performance Power Plan — the biggest single win:

  1. Search Command Prompt → Run as Administrator
  2. Run: powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
  3. Open Power Options → select Ultimate Performance

2. Disable unnecessary services — frees 150–400MB RAM:

  1. Press Win + R → type services.msc
  2. Find and set these to Disabled (right-click → Properties → Startup type):
ServiceActionRAM saved
Xbox Live Auth ManagerDisabled~30MB
Xbox Live Game SaveDisabled~25MB
Xbox Live Networking ServiceDisabled~30MB
FaxDisabled~20MB
Print Spooler (if no printer)Disabled~40MB

3. Adjust visual effects for performance:

  1. Press Win + R → type sysdm.cpl → Advanced → Performance Settings
  2. Select "Adjust for best performance"
  3. Re-enable: "Smooth edges of screen fonts" only
📈
Combined effect: +10–20fps, 200–400MB extra RAM free for VALORANT On 8GB RAM systems this is the difference between hitting page file (stutter) and staying in RAM.
Section 08 What NOT to Do — Common Mistakes That Hurt Performance

These are the most common "optimisations" shared online that either do nothing or actively hurt performance on low-end hardware:

MistakeWhy It's Wrong
Disabling CPU cores VALORANT uses multiple cores. Disabling them reduces available threads and hurts performance, not helps it.
Setting CPU affinity to specific cores Windows already manages this better than manual assignment. Forcing affinity creates scheduling conflicts.
Overclocking RAM beyond XMP profile Manual overclocks beyond rated speed often cause instability and crashes in VALORANT. Stick to XMP.
Using "Fullscreen Optimisations" disabled Outdated advice from 2019. On modern Windows 11 builds this setting no longer affects VALORANT performance.
MSI Afterburner OSD during play The overlay uses CPU cycles. Use VALORANT's built-in Shift+F1 stats instead.
Deleting pagefile entirely Some guides suggest this. On 8GB systems it causes crashes. Leave the pagefile on — Section 7 of our stuttering guide shows how to size it correctly.
⚠️
If a guide tells you to edit bcdedit or disable Windows Defender completely — skip it. These carry real security risks and provide no measurable FPS gain in VALORANT.
