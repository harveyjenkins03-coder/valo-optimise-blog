Fix 01 Identify Your Crash Type First

Not all crashes are the same. The fix for a GPU driver timeout is completely different from a Vanguard block or memory leak.

Here is how to identify your crash type:

  1. Press Win + X, open Event Viewer
  2. Go to Windows Logs > Application
  3. Look for Error entries around your crash time
  4. Note the Source column
  5. Check %localappdata%\VALORANT\Saved\Logs for game logs

Fix 02 Update GPU Drivers with DDU Clean Install

GPU driver crashes are the #1 cause of VALORANT crashing. Symptoms: black screen, 2–5 second freeze, then desktop. Event Viewer shows Display driver stopped responding.

The problem is leftover files from old driver installations conflicting with new ones.

  1. Download DDU from guru3d.com
  2. Download latest stable driver from NVIDIA/AMD
  3. Boot into Safe Mode
  4. Run DDU > Clean and restart
  5. Install fresh driver after reboot
  6. Restart once more

NVIDIA: Choose Custom > Clean installation. Skip GeForce Experience if you do not use ShadowPlay.

AMD: Use AMD Cleanup Utility first. Enable Anti-Lag, disable Radeon Chill and Enhanced Sync.

✓
Clean driver installs fix 60–70% of GPU-related crashes Even if drivers are current, a clean install resolves hidden conflicts.
Fix 03 Fix Vanguard Anti-Cheat Errors

Riot Vanguard runs at boot and has strict requirements. If not met, it blocks or crashes the game.

ErrorMeaningFix
VAN 9001Secure Boot disabledEnable in BIOS
VAN 9003TPM 2.0 missingEnable fTPM/PTT in BIOS
VAN 1067Not initializedFull restart (not sleep)
VAN -81Service errorReinstall Vanguard
  1. services.msc > find vgc > set Running + Automatic
  2. Enter BIOS (Del or F2 at boot)
  3. Enable Secure Boot
  4. Enable TPM (Intel PTT / AMD fTPM)
  5. Save and exit
  6. If still broken: uninstall Vanguard, relaunch VALORANT to reinstall
⚠️
Never disable Vanguard to fix crashes. The issue is system config, not Vanguard. Disabling it prevents playing entirely.
Fix 04 Repair DirectX and Visual C++ Redistributables

VALORANT needs DirectX 11 and Visual C++ packages. Corrupted or missing ones cause silent crashes or Unreal Engine errors.

Common after Windows major updates that overwrite runtime libraries.

  1. Install DirectX End-User Runtime from Microsoft
  2. Install all VC++ Redistributables (2015–2022, x86 + x64)
  3. Restart PC
  4. Test VALORANT

Still crashing? Run the .NET Framework Repair Tool.

✓
Fixes the silent crash — VALORANT closes instantly with no error popup
Fix 05 Test and Fix Faulty RAM

Faulty RAM produces completely random crashes. No pattern — which is exactly what makes it hard to diagnose.

  1. Win + R > mdsched.exe > Restart and check
  2. Wait 10–20 minutes for Memory Diagnostic
  3. Check Event Viewer > System for MemoryDiagnostics-Results
  4. If errors: remove sticks one at a time to find the bad one

For thorough testing, use MemTest86 (bootable USB, 4+ passes).

Check RAM speed: Task Manager > Performance > Memory. If 3200MHz shows 2133MHz, enable XMP/EXPO in BIOS.

⚠️
RAM errors worsen over time. Even one MemTest86 error means replace that stick. Faulty RAM corrupts data system-wide.
Fix 06 Fix Overheating Crashes

When CPU/GPU hits thermal limits, it throttles or shuts off. VALORANT shows a freeze then crash to desktop or full shutdown.

  1. Download HWiNFO64 (free)
  2. Run alongside VALORANT for a full match
  3. Check CPU and GPU max temps
  4. Above 90 degrees = thermal problem

Overheating fixes:

  1. Clean dust from fans/heatsinks with compressed air
  2. Reapply thermal paste if 2+ years old
  3. Ensure at least one intake + one exhaust fan
  4. Laptops: cooling pad + elevate rear
  5. Undervolt CPU with ThrottleStop or Ryzen Master
✓
Dropping from 95 to 75 degrees eliminates crashes AND boosts FPS 15–20%
Fix 07 Disable GPU Overclocks

VALORANT is more sensitive to clock instability than most games.

  1. Open MSI Afterburner
  2. Reset to stock clocks
  3. Power Limit 100%
  4. Test 3+ matches

If stable at stock, try 25-50MHz lower next time.

⚠️
VRAM overclocks are harder to detect. Causes artifacts then crashes 10-30 min later.
Fix 08 Update Windows Fully

Outdated Windows = missing Vanguard patches + known DirectX bugs.

  1. Settings > Windows Update > Check
  2. Install all including optional
  3. Advanced > Optional for drivers
  4. Restart and recheck

Run winver. If older than 23H2 on Win 11, update now.

✓
Windows 11 24H2+ includes GPU stability fixes
Fix 09 Repair System Files (SFC/DISM)

Corrupted system files break DirectX, drivers, and memory management.

sfc /scannow

Then:

DISM /Online /Cleanup-Image /RestoreHealth

If DISM fixes anything, run sfc again.

✓
Fixes crashes from Windows Update corruption
Fix 10 Disable Fullscreen Optimisations

Windows 11 forces hybrid windowed mode which can crash VALORANT.

  1. Find VALORANT-Win64-Shipping.exe
  2. Properties > Compatibility
  3. Check Disable fullscreen optimizations
  4. Change high DPI > Override (Application)
  5. Apply
✓
Also reduces input latency by 1-3ms
Fix 11 Fix Virtual Memory (Page File)

Small or disabled page file crashes VALORANT during heavy scenes.

  1. sysdm.cpl > Advanced > Performance > Settings
  2. Advanced > Virtual Memory > Change
  3. Uncheck auto manage
  4. Select fastest drive
  5. Custom: Initial = RAM x 1.5, Max = RAM x 3
  6. Set > OK > Restart

⚠️
Never disable page file entirely. Vanguard expects it to exist.
